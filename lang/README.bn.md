একটি শক্তিশালী ওয়েব ক্রলিং টুল যা MCP (Model Context Protocol)-এর মাধ্যমে AI সহকারীদের সাথে একীভূত হয়। এই প্রকল্পটি AI সহকারীদের ওয়েবসাইট ক্রল করতে, গতিশীল কন্টেন্ট এক্সট্র্যাক্ট করতে, লিঙ্কের মাধ্যমে নেভিগেট করতে এবং সরাসরি কাঠামোগত Markdown ফাইল সংরক্ষণ করতে সক্ষম করে।

## 📋 বৈশিষ্ট্যসমূহ

- MCP-এর মাধ্যমে AI সহকারীদের সাথে নেটিভ একীকরণ
- স্ক্র্যাপ করা Markdown কন্টেন্ট সরাসরি AI-তে ফেরত দেওয়া
- AI নেভিগেশনের জন্য অভ্যন্তরীণ/বাহ্যিক লিঙ্ক এক্সট্র্যাক্ট এবং প্রদর্শন করা
- কনফিগারযোগ্য গভীরতার সাথে ওয়েবসাইট ক্রলিং
- ক্রল ফলাফলের বিস্তারিত পরিসংখ্যান
- ত্রুটি এবং পাওয়া যায়নি এমন পৃষ্ঠার হ্যান্ডলিং
- **উন্নত স্ক্র্যাপিং সক্ষমতা**:
  - **ম্যাজিক মোড**: অ্যান্টি-বট (যেমন Cloudflare) বাইপাস করুন এবং বাস্তব ব্রাউজার আচরণ সিমুলেট করুন
  - **টার্গেটেড এক্সট্র্যাকশন**: CSS সিলেক্টর ব্যবহার করে শুধুমাত্র আপনার প্রয়োজনীয় বিষয়বস্তু আনুন
  - **কাস্টম জাভাস্ক্রিপ্ট**: এক্সট্র্যাকশনের আগে কোড এক্সিকিউট করুন (ক্লিক, স্ক্রোল, ফর্ম পূরণ)
  - **স্থায়ী সেশন**: প্রমাণীকৃত সাইটের জন্য রিকোয়েস্ট জুড়ে কুকিজ এবং স্টেট বজায় রাখুন
  - **SPA সমর্থন**: গতিশীল CSS সিলেক্টরের জন্য অপেক্ষা করুন অথবা স্পষ্ট প্রি-এক্সট্র্যাকশন বিলম্ব নির্ধারণ করুন

## 🚀 MCP কনফিগারেশন

এই টুল ব্যবহারের সবচেয়ে সহজ এবং প্রস্তাবিত উপায় হল `uvx`-এর মাধ্যমে, যা স্বয়ংক্রিয়ভাবে PyPI থেকে সর্বশেষ প্রকাশিত সংস্করণ আনে এবং চালায়।

### পূর্বশর্ত

- আপনার সিস্টেমে [uv](https://github.com/astral-sh/uv) ইনস্টল থাকতে হবে।

### AI সহকারীদের জন্য সেটআপ (যেমন, Claude Desktop, Cline)

নিম্নলিখিতটি আপনার AI সহকারীর MCP কনফিগারেশন ফাইলে (যেমন, `cline_mcp_settings.json` বা `claude_desktop_config.json`) যোগ করুন:

> **Windows ব্যবহারকারীদের জন্য নোট**: কিছু নির্ভরতার সাথে কম্পাইলেশন সমস্যা এড়াতে `--python 3.13` নির্দিষ্ট করা অত্যন্ত সুপারিশ করা হয়।

```json
{
  "mcpServers": {
    "crawl": {
      "command": "uvx",
      "args": [
        "--python",
        "3.13",
        "crawl4ai-mcp-llm"
      ],
      "disabled": false,
      "autoApprove": [],
      "timeout": 600
    }
  }
}
```

### গুরুত্বপূর্ণ: ব্রাউজার ইনস্টলেশন

ক্রলারটি গতিশীল কন্টেন্ট হ্যান্ডল করতে Playwright ব্যবহার করে। টুল সেট আপ করার পরে আপনাকে অবশ্যই প্রয়োজনীয় ব্রাউজারগুলি ইনস্টল করতে হবে:

```bash
uv run playwright install chromium
```

## 🖥️ ব্যবহার

একবার কনফিগার করা হলে, আপনি আপনার AI সহকারীকে একটি ক্রল সম্পাদনের জন্য অনুরোধ করে ক্রলারটি ব্যবহার করতে পারেন।

### Claude/Cline এর সাথে ব্যবহারের উদাহরণ

- **সাধারণ ক্রল**: "আপনি কি example.com সাইটটি ক্রল করে আমাকে একটি সারাংশ দিতে পারেন?"
- **অপশনসহ ক্রল**: "আপনি কি https://example.com কে 3 গভীরতায় ক্রল করতে পারেন এবং বাহ্যিক লিঙ্ক অন্তর্ভুক্ত করতে পারেন?"
- **গতিশীল কন্টেন্ট**: "এই React অ্যাপটি ক্রল করুন এবং `.main-content` সিলেক্টর লোড হওয়া পর্যন্ত অপেক্ষা করুন।"
- **সুরক্ষা বাইপাস**: "example.com ক্রল করুন কিন্তু অ্যান্টি-বট সুরক্ষা বাইপাস করতে 'magic mode' ব্যবহার করুন।"
- **টার্গেটেড এক্সট্র্যাকশন**: "ডক্স সাইটটি ক্রল করুন কিন্তু শুধুমাত্র `h1, p.lead` CSS সিলেক্টরের সাথে মিলে যাওয়া কন্টেন্ট এক্সট্র্যাক্ট করুন।"

## 🛠️ উপলব্ধ প্যারামিটার (MCP টুল)

`crawl` টুলটি নিম্নলিখিত প্যারামিটারগুলি গ্রহণ করে:

| প্যারামিটার | টাইপ | বিবরণ | ডিফল্ট মান |
|-----------|------|-------------|---------------|
| `url` | string | ক্রল করার জন্য URL (আবশ্যক) | - |
| `max_depth` | integer | সর্বোচ্চ ক্রলিং গভীরতা | 2 |
| `include_external` | boolean | বাহ্যিক লিঙ্ক অন্তর্ভুক্ত করুন | false |
| `verbose` | boolean | বিস্তারিত আউটপুট সক্ষম করুন | true |
| `wait_for_selector` | string | কন্টেন্ট এক্সট্র্যাক্ট করার আগে যে CSS সিলেক্টরের জন্য অপেক্ষা করতে হবে। সিঙ্গেল-পেজ অ্যাপ্লিকেশনের জন্য কার্যকর। | None |
| `return_content` | boolean | এক্সট্র্যাক্ট করা কন্টেন্ট সরাসরি MCP রেসপন্সে ফেরত দেবে কিনা (প্রয়োজনে 50k অক্ষরে ট্রাঙ্কেট করা হয়)। | true |
| `output_file` | string | আউটপুট ফাইলের পাথ | স্বয়ংক্রিয়ভাবে তৈরি হয় |
| `magic` | boolean | অ্যান্টি-বট বাইপাস করতে এবং বাস্তব ব্রাউজার সিমুলেট করতে ম্যাজিক মোড সক্ষম করুন | false |
| `css_selector` | string | পৃষ্ঠা থেকে শুধুমাত্র টার্গেটেড এলিমেন্ট এক্সট্র্যাক্ট করতে নির্দিষ্ট CSS সিলেক্টর | None |
| `js_code` | string | এক্সট্র্যাকশনের আগে পৃষ্ঠায় এক্সিকিউট করার জন্য কাস্টম জাভাস্ক্রিপ্ট কোড | None |
| `session_id` | string | রিকোয়েস্ট জুড়ে কুকিজ এবং ব্রাউজার স্টেট বজায় রাখতে স্থায়ী সেশন শনাক্তকারী | None |
| `delay_before_return_html` | number | HTML এক্সট্র্যাক্ট করার আগে অপেক্ষা করার সময় সেকেন্ডে (ভারী JS পৃষ্ঠার জন্য কার্যকর) | None |
