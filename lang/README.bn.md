একটি শক্তিশালী ওয়েব ক্রলিং টুল যা MCP (Model Context Protocol) এর মাধ্যমে এআই (AI) অ্যাসিস্ট্যান্টদের সাথে একীভূত হয়। এই প্রকল্পটি এআই অ্যাসিস্ট্যান্টদের ওয়েবসাইট ক্রল করতে, ডায়নামিক কন্টেন্ট এক্সট্র্যাক্ট করতে, লিঙ্কের মাধ্যমে নেভিগেট করতে এবং সরাসরি স্ট্রাকচার্ড মার্কডাউন ফাইল সংরক্ষণ করতে সাহায্য করে।

## 📋 বৈশিষ্ট্যসমূহ

- MCP এর মাধ্যমে এআই অ্যাসিস্ট্যান্টদের সাথে নেটিভ ইন্টিগ্রেশন
- স্ক্র্যাপ করা মার্কডাউন কন্টেন্ট সরাসরি এআই-এর কাছে ফেরত পাঠায়
- এআই নেভিগেশনের জন্য ইন্টারনাল/এক্সটারনাল লিঙ্ক এক্সট্র্যাক্ট এবং প্রদর্শন করে
- কন্টেন্ট এক্সট্র্যাক্ট করার আগে ডায়নামিক CSS সিলেক্টরের জন্য অপেক্ষা করে (SPA সাপোর্ট)
- কনফিগারেবল ডেপথ সহ ওয়েবসাইট ক্রলিং
- বিস্তারিত ক্রল রেজাল্ট স্ট্যাটিস্টিকস
- এরর এবং নট ফাউন্ড পেজ হ্যান্ডলিং

## 🚀 MCP কনফিগারেশন

এই টুলটি ব্যবহার করার সবচেয়ে সহজ এবং প্রস্তাবিত উপায় হল `uvx` এর মাধ্যমে, যা ম্যানুয়ালি রিপোজিটরি ক্লোন করার প্রয়োজন ছাড়াই স্বয়ংক্রিয়ভাবে GitHub থেকে লেটেস্ট ভার্সন সংগ্রহ করে এবং চালায়।

### প্রয়োজনীয় শর্তাবলী

- আপনার সিস্টেমে [uv](https://github.com/astral-sh/uv) ইনস্টল থাকতে হবে।

### এআই অ্যাসিস্ট্যান্টদের জন্য সেটআপ (যেমন Claude Desktop, Cline)

আপনার এআই অ্যাসিস্ট্যান্টের MCP কনফিগারেশন ফাইলে (যেমন `cline_mcp_settings.json` বা `claude_desktop_config.json`) নিচের অংশটি যোগ করুন:

> **উইন্ডোজ ব্যবহারকারীদের জন্য নোট**: নির্দিষ্ট ডিপেন্ডেন্সিগুলোর সাথে কম্পাইলেশন সমস্যা এড়াতে `--python 3.12` উল্লেখ করার জন্য দৃঢ়ভাবে সুপারিশ করা হচ্ছে।

```json
{
  "mcpServers": {
    "crawl": {
      "command": "uvx",
      "args": [
        "--python",
        "3.12",
        "--from",
        "git+https://github.com/laurentvv/crawl4ai-mcp",
        "crawl4ai-mcp"
      ],
      "disabled": false,
      "autoApprove": [],
      "timeout": 600
    }
  }
}
```

### গুরুত্বপূর্ণ: ব্রাউজার ইনস্টলেশন

ক্রলারটি ডায়নামিক কন্টেন্ট হ্যান্ডেল করার জন্য Playwright ব্যবহার করে। টুলটি সেটআপ করার পরে আপনাকে অবশ্যই প্রয়োজনীয় ব্রাউজারগুলো ইনস্টল করতে হবে:

```bash
uv run playwright install chromium
```

## 🖥️ ব্যবহার পদ্ধতি

কনফিগার করার পরে, আপনি আপনার এআই অ্যাসিস্ট্যান্টকে ক্রল করার জন্য অনুরোধ করে ক্রলারটি ব্যবহার করতে পারেন।

### Claude/Cline এর সাথে ব্যবহারের উদাহরণ

- **সাধারণ ক্রল**: "তুমি কি example.com সাইটটি ক্রল করে আমাকে একটি সারাংশ দিতে পারো?"
- **অপশন সহ ক্রল**: "তুমি কি https://example.com সাইটটি ৩ ডেপথ পর্যন্ত ক্রল করতে পারো এবং এক্সটারনাল লিঙ্কগুলো অন্তর্ভুক্ত করতে পারো?"
- **ডায়নামিক কন্টেন্ট**: "এই React অ্যাপটি ক্রল করো এবং `.main-content` সিলেক্টর লোড হওয়া পর্যন্ত অপেক্ষা করো।"

## 🛠️ উপলব্ধ প্যারামিটার (MCP টুল)

`crawl` টুলটি নিচের প্যারামিটারগুলো গ্রহণ করে:

| প্যারামিটার | টাইপ | বর্ণনা | ডিফল্ট ভ্যালু |
|-----------|------|-------------|---------------|
| `url` | string | ক্রল করার জন্য URL (প্রয়োজনীয়) | - |
| `max_depth` | integer | সর্বোচ্চ ক্রলিং ডেপথ | 2 |
| `include_external` | boolean | এক্সটারনাল লিঙ্ক অন্তর্ভুক্ত করা | false |
| `verbose` | boolean | বিস্তারিত আউটপুট সক্ষম করা | true |
| `wait_for_selector` | string | কন্টেন্ট এক্সট্র্যাক্ট করার আগে যে CSS সিলেক্টরের জন্য অপেক্ষা করতে হবে। সিঙ্গেল পেজ অ্যাপ্লিকেশনের (SPA) জন্য দরকারী। | None |
| `return_content` | boolean | এক্সট্র্যাক্ট করা কন্টেন্ট সরাসরি MCP রেসপন্সে ফেরত পাঠানো হবে কিনা (প্রয়োজনে ৫০ হাজার ক্যারেক্টার পর্যন্ত ছোট করা হবে)। | true |
| `output_file` | string | আউটপুট ফাইল পাথ | স্বয়ংক্রিয়ভাবে তৈরি হয় |

## 👨‍💻 ডেভেলপমেন্ট

আপনি যদি ক্রলারটি পরিবর্তন করতে চান বা লোকাললি চালাতে চান:

1. এই রিপোজিটরিটি ক্লোন করুন:
```bash
git clone https://github.com/laurentvv/crawl4ai-mcp
cd crawl4ai-mcp
```

2. `uv` ব্যবহার করে ডিপেন্ডেন্সি ইনস্টল করুন:
```bash
uv sync
```

3. সরাসরি MCP সার্ভার চালান:
```bash
uv run crawl4ai-mcp
```
