एक शक्तिशाली वेब क्रॉलिंग टूल जो MCP (Model Context Protocol) के माध्यम से AI सहायकों के साथ एकीकृत होता है। यह प्रोजेक्ट AI सहायकों को वेबसाइट्स को क्रॉल करने, डायनामिक कंटेंट निकालने, लिंक्स के माध्यम से नेविगेट करने और सीधे संरचित Markdown फ़ाइलों को सहेजने की अनुमति देता है।

## 📋 विशेषताएँ

- MCP के माध्यम से AI सहायकों के साथ नेटिव एकीकरण
- स्क्रैप की गई Markdown कंटेंट सीधे AI को लौटाना
- AI नेविगेशन के लिए आंतरिक/बाहरी लिंक्स निकालना और प्रदर्शित करना
- कॉन्फ़िगर करने योग्य गहराई के साथ वेबसाइट क्रॉलिंग
- विस्तृत क्रॉल परिणाम आँकड़े
- त्रुटि और न मिले पेजों को संभालना
- **उन्नत स्क्रैपिंग क्षमताएँ**:
  - **मैजिक मोड**: एंटी-बॉट्स (जैसे Cloudflare) को बायपास करें और वास्तविक ब्राउज़र व्यवहार का अनुकरण करें
  - **लक्षित निष्कर्षण**: CSS सिलेक्टर्स का उपयोग करके केवल आवश्यक चीज़ें प्राप्त करें
  - **कस्टम JavaScript**: निष्कर्षण से पहले कोड निष्पादित करें (क्लिक, स्क्रॉल, फ़ॉर्म भरना)
  - **स्थायी सत्र**: प्रमाणित साइटों के लिए अनुरोधों के बीच कुकीज़ और स्थिति बनाए रखें
  - **SPA समर्थन**: डायनामिक CSS सिलेक्टर्स की प्रतीक्षा करें या स्पष्ट पूर्व-निष्कर्षण विलंब सेट करें

## 🚀 MCP कॉन्फ़िगरेशन

इस टूल का उपयोग करने का सबसे सरल और अनुशंसित तरीका `uvx` के माध्यम से है, जो स्वचालित रूप से PyPI से नवीनतम प्रकाशित संस्करण प्राप्त करता है और चलाता है।

### पूर्वापेक्षाएँ

- आपके सिस्टम पर [uv](https://github.com/astral-sh/uv) स्थापित होना चाहिए।

### AI सहायकों के लिए सेटअप (उदा., Claude Desktop, Cline)

अपने AI सहायक की MCP कॉन्फ़िगरेशन फ़ाइल में निम्नलिखित जोड़ें (उदा., `cline_mcp_settings.json` या `claude_desktop_config.json`):

> **Windows उपयोगकर्ताओं के लिए नोट**: कुछ डिपेंडेंसीज़ के साथ संकलन संबंधी समस्याओं से बचने के लिए `--python 3.13` निर्दिष्ट करने की दृढ़ता से अनुशंसा की जाती है।

```json
{
  "mcpServers": {
    "crawl": {
      "command": "uvx",
      "args": [
        "--python",
        "3.13",
        "crawl4ai-mcp-llm"
      ],
      "disabled": false,
      "autoApprove": [],
      "timeout": 600
    }
  }
}
```

### महत्वपूर्ण: ब्राउज़र स्थापना

क्रॉलर डायनामिक कंटेंट को संभालने के लिए Playwright का उपयोग करता है। टूल सेटअप करने के बाद आपको आवश्यक ब्राउज़र इंस्टॉल करने होंगे:

```bash
uv run playwright install chromium
```

## 🖥️ उपयोग

एक बार कॉन्फ़िगर हो जाने पर, आप अपने AI सहायक से क्रॉल करने का अनुरोध करके क्रॉलर का उपयोग कर सकते हैं।

### Claude/Cline के साथ उपयोग के उदाहरण

- **सरल क्रॉल**: "क्या आप example.com साइट को क्रॉल कर सकते हैं और मुझे सारांश दे सकते हैं?"
- **विकल्पों के साथ क्रॉल**: "क्या आप https://example.com को 3 की गहराई के साथ क्रॉल कर सकते हैं और बाहरी लिंक्स शामिल कर सकते हैं?"
- **डायनामिक कंटेंट**: "इस React ऐप को क्रॉल करें और `.main-content` सिलेक्टर के लोड होने की प्रतीक्षा करें।"
- **सुरक्षा बायपास**: "example.com को क्रॉल करें लेकिन एंटी-बॉट सुरक्षा को बायपास करने के लिए 'मैजिक मोड' का उपयोग करें।"
- **लक्षित निष्कर्षण**: "डॉक्स साइट को क्रॉल करें लेकिन केवल `h1, p.lead` CSS सिलेक्टर से मेल खाने वाला कंटेंट निकालें।"

## 🛠️ उपलब्ध पैरामीटर (MCP टूल)

`crawl` टूल निम्नलिखित पैरामीटर स्वीकार करता है:

| पैरामीटर | प्रकार | विवरण | डिफ़ॉल्ट मान |
|-----------|------|-------------|---------------|
| `url` | string | क्रॉल करने के लिए URL (आवश्यक) | - |
| `max_depth` | integer | अधिकतम क्रॉलिंग गहराई | 2 |
| `include_external` | boolean | बाहरी लिंक्स शामिल करें | false |
| `verbose` | boolean | विस्तृत आउटपुट सक्षम करें | true |
| `wait_for_selector` | string | कंटेंट निकालने से पहले प्रतीक्षा करने के लिए CSS सिलेक्टर। सिंगल-पेज एप्लिकेशन के लिए उपयोगी। | None |
| `return_content` | boolean | निकाले गए कंटेंट को सीधे MCP प्रतिक्रिया में लौटाना है या नहीं (आवश्यक होने पर 50k वर्णों तक सीमित)। | true |
| `output_file` | string | आउटपुट फ़ाइल पथ | स्वचालित रूप से उत्पन्न |
| `magic` | boolean | एंटी-बॉट्स को बायपास करने और वास्तविक ब्राउज़र का अनुकरण करने के लिए मैजिक मोड सक्षम करें | false |
| `css_selector` | string | पेज से केवल लक्षित तत्व निकालने के लिए विशिष्ट CSS सिलेक्टर | None |
| `js_code` | string | निष्कर्षण से पहले पेज पर निष्पादित करने के लिए कस्टम JavaScript कोड | None |
| `session_id` | string | अनुरोधों के बीच कुकीज़ और ब्राउज़र स्थिति बनाए रखने के लिए स्थायी सत्र पहचानकर्ता | None |
| `delay_before_return_html` | number | HTML निकालने से पहले प्रतीक्षा करने के लिए सेकंड में विलंब (भारी JS पेजों के लिए उपयोगी) | None |
